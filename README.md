1일 최소 1문제 업로드 OR JS 이론 책읽은 지식 정리

# 커밋 규칙

### Format

{Keyword}: {출처} {제목}

### Keyword

- solve: 풀이에 성공한 문제
- fail: 풀이에 실패한 문제
- improve: 새로운 성공 풀이를 추가한 문제
- docs: readme 등 문서 업데이트
- cs: javaScript CS 문제 및 풀이

# 파일 구조

```bash
├── 출처
│   ├── 제목
│   │    └── JS 파일
├── CS
│   └── MD 파일

```

- JS 파일 안에 **코드**와 **설명**을 추가!
