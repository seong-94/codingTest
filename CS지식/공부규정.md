# 공부 규정

문제의 요점을 제대로 파악하지 못하고 문제를 접근하는 방식은 너무 비효율 적 이다.

우선 문제를 풀지만, 문제 보단 JavaScript CS 지식과 코테 에 대한 공부를 우선 으로 하는것으로 생각 해야할거같다.
