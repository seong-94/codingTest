## JS 121번 무엇이 출력 될까요?

```
const config = {
  languages: [],
  set language(lang) {
    return this.languages.push(lang);
  },
};

console.log(config.language);
```

- A: function language(lang) { this.languages.push(lang }
- B: 0
- C: []
- D: undefined

<details>
<summary>정답</summary>
<div>

답: D

setter 메소드 는 실제 값을 반환하지 않는다. setter 의 주요기능은 `속성` 을 수정 하는 것입니다.
따라서 undefined 가 반환됩니다.
